Student Internship on Cyber Security

  • During the internship, I explored various topics related to web application security, focusing on OWASP (Open Web Application Security Project) guidelines, which highlight the top vulnerabilities like SQL injection, cross-site scripting (XSS), and insecure deserialization. These topics provided a foundation for understanding the most common and critical security flaws found in web applications. Additionally, TryHackMe courses introduced practical exercises for cybersecurity, offering hands-on labs to strengthen your skills in detecting and mitigating vulnerabilities in real-world scenarios.
  • Burp Suite, a professional tool for penetration testing, was a key focus. This allowed me to understand how attackers might manipulate data and probe for weaknesses in web applications. I also learned how to use Burp Suite for automated scanning, analyzing traffic, and testing for security misconfigurations.
  • Other techniques, such as Google Dorking, were covered, helping me to refine search queries to locate sensitive information that might be unintentionally exposed online. The combination of theoretical knowledge and practical assignments allowed me to develop a strong grasp of vulnerability detection and reporting, aligning my skills with industry standards in web application security.
